This is my first Ruby on Rails application:
A Pinterest-like website from my Rails course One Month Rails. The idea was to create a fully functional early version of pinterest.
Users can create an account, upload starwars images, and share them with friends from the starwars fan community.

I will try to update it as much as possible to make it as close to the current version of pinterest.
